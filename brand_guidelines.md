# Qualz.ai Brand Guidelines for Presentations

## Color Palette

### Primary Colors
- **Primary Purple**: #7C3AED
  - Use for: Main headings, primary buttons, borders, accents
- **Secondary Green**: #10B981
  - Use for: Success elements, checkmarks, secondary accents, highlights
- **Accent Orange/Yellow**: #F59E0B
  - Use for: Call-to-action elements, important notices, tertiary accents
- **Text Dark Gray**: #111827
  - Use for: Body text, content

### Background Colors
- **White**: #FFFFFF
- **Light Purple Gradient**: linear-gradient(135deg, #FFFFFF 0%, #F5F3FF 100%)
  - Use for: Slide backgrounds

### Supporting Colors
- **Blue**: #3B82F6
  - Use for: Informational elements, links, tertiary accents
- **Light Gray**: #F3F4F6
  - Use for: Subtle backgrounds, dividers
- **Medium Gray**: #4B5563
  - Use for: Subtitles, secondary text

## Typography

### Font Family
- Primary Font: Arial, Helvetica, sans-serif
  - A clean, modern sans-serif font that works well across all platforms

### Font Sizes
- **Front Page**:
  - Headline: 72px
  - Subtitle: 36px
  - Body: 24px

- **Content Pages**:
  - Headline: 36px
  - Subtitle: 24px
  - Body: 20px

### Font Weights
- Headings: Bold (700)
- Subheadings: Medium (500)
- Body Text: Regular (400)

## Layout Guidelines

### Slide Dimensions
- **Aspect Ratio**: 16:9 (1280px × 720px)
- **Margins**: 60px from edges

### Logo Placement
- Top left corner
- Size: 40px height (maintain aspect ratio)
- Padding: 20px from top and left edges

### Content Structure
- Clear visual hierarchy with distinct sections
- Use white space effectively
- Limit content to essential information
- Use color-coded elements to organize information

## Design Elements

### Cards & Boxes
- White background
- Subtle shadow (0 4px 6px rgba(0, 0, 0, 0.05))
- Rounded corners (8px border-radius)
- Color-coded borders (5px) on left or top

### Icons
- Font Awesome icons
- Color-coded to match brand palette
- Use consistently for similar elements

### Visual Hierarchy
1. Page title (largest, purple)
2. Section headings (medium, color-coded)
3. Body content (smallest, dark gray)

## Best Practices

1. **Consistency**: Maintain consistent use of colors, fonts, and spacing
2. **Simplicity**: Keep slides clean and focused on key messages
3. **White Space**: Use ample white space to improve readability
4. **Color Coding**: Use colors consistently to indicate different types of information
5. **Visual Balance**: Create balanced layouts with proper alignment
6. **Readability**: Ensure text is large enough and has sufficient contrast
7. **Branding**: Always include the Qualz.ai logo on every slide

## File Formats

- Save presentations in standard formats (PPTX, PDF)
- Maintain 16:9 aspect ratio for all slides
- Ensure all text is editable (not flattened into images)

---

These guidelines will help maintain a consistent, professional brand identity across all Qualz.ai presentation materials.
